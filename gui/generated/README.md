# gui/generated

Qt `.ui` layouts generated by `tools/lfm2qt` from Cheat Engine's `.lfm` forms,
used as the reference for making our GUI look like CE and for tracking feature
parity.

The `*.ui` files here are **gitignored** — they are mechanically derived from an
external CE checkout, so we don't commit them. Regenerate them with:

```sh
python3 tools/lfm2qt/lfm2qt.py \
    --ce-source "<path-to-CE-checkout>/Cheat Engine" \
    --out-ui gui/generated \
    --out-report docs/CE_FEATURE_INVENTORY.md
```

To wire a generated form into the app: run `uic <form>.ui -o ui_<form>.h`, then
write a small `QWidget`/`QMainWindow` subclass that calls `setupUi()` and connects
the controls to our own backend (never CE's Pascal logic).
